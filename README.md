# Naviotech_Project1_California_House_Price_Prediction_Project
This repository consists of  my project of California House Pricing Dataset Analysis along with Streamlit Predictor. The prediction has been made using the Random Forest Regressor.  

The prediction is made on 8 parameters: 
1. Median Income
2. house Age
3. Average Rooms
4. Average Bedrooms
5. Population
6. Average Occupancy
7. Latitude
8. Longitude

Here, a user interface has been made using the Streamlit app: https://cmangla-house-price-predictor.streamlit.app/
